# 📊 Social Media Advertising Analytics

## Project Overview

This is a **GitHub-safe version** of the Social Media Advertising Analytics project.

The original CSV file is large, so this repository includes a smaller sample dataset that GitHub can accept easily.

---

## Business Problem

Marketing teams need to understand which advertising campaigns, channels, customer segments, locations, and campaign goals produce the best return on investment.

This project helps answer:

- Which channel gives the best ROI?
- Which customer segment performs best?
- Which campaign goal should receive more budget?
- Which markets and languages perform better?
- Which campaigns are likely to become high ROI campaigns?

Install required libraries:

```bash

```

Open Jupyter Notebook:

```bash

```

Then run:

```text

```
